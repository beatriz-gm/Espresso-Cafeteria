# ☕ Espresso Cafeteria

Um website responsivo desenvolvido como atividade da disciplina de Programação Front-End do curso de Engenharia de Software.

O projeto simula o site de uma cafeteria fictícia, apresentando um cardápio interativo, página institucional e formulário de contato com validação em JavaScript.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## ✨ Funcionalidades

- 🏠 Página inicial com filtro mostrando Destaques da Semana
- 📖 Cardápio dinâmico
- 🔍 Filtro de produtos por categoria
- 📱 Layout responsivo
- 📧 Formulário de contato com validação
- ⬆️ Botão "Voltar ao topo"
- 🔎 SEO básico (meta description, viewport e uso de HTML semântico)
